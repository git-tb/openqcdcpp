Detailed Description

template<typename T>
class ProtectedObject< T >

ProtectedObjects are those that should not be changed everywhere in the code. On one hand, we could just declare constant global variables at the beginning of each program. Alternatively we can keep them non-const but restrict assignment by making them private in a dedicated class. This simplifies static code analysis.
